# HadoopLog
学习使用hive，hadoop日志清洗之后导入mysql

主要工具：
  IntelliJ Idea 2017
  hadoop集群（3台centos6.5、virtualbox）
  Hive
  Mysql
  
实验流程：
  在本地机器（MacOX）的idea上编写java代码（主要是JDBC 操纵Hive、操纵Mysql）,远程操作虚拟机中的hadoop、hive、mysql。


遇到了很多坑，有兴趣的可以研究一下，当然也欢迎联系我一同探讨
